O que é o Ultron?

Trata-se de um sistema para gestão de ações (compras e vendas) com base nas informações obtidas diretamente da bolsa de valores.

Como participar deste sistema?

Para obter acesso ao Ultron é necessário que um dos admnistradores aceite sua solicitação e forneça os dados para login no sistema.

Tipos de Usuários

  • Investidor
  • Gestor
  • Admnistrador
